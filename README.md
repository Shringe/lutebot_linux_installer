# lutebot_linux_installer
Small set of scripts to get [LuteMod + LuteBot](https://github.com/Dimencia/LuteBot3) working on Linux

# LuteMod
This is whats used to play songs in game.
## Installation
Edit the `STEAM_LIBRARY` path if Mordhau is installed on another steam library, and then run `./lutemod_install.sh`.

# LuteBot
This lets you upload songs for LuteMod to play.
## Usage
Make sure to have [protontricks](https://github.com/Matoking/protontricks) installed. After Mordhau is launched, run `./lutebot_run.sh` to open LuteBot. If it prompts to install LuteMod again, click deny.
